# 🎓 College & Student Registration System

A simple web-based College Login and Student Registration system built using:

- HTML5  
- CSS3  
- JavaScript  
- LocalStorage (for data persistence)  

This project demonstrates frontend form handling, validation, page connection, and basic authentication logic without using any backend.

---

## 📌 Features

### 🏫 College Login Page
- College Login ID
- Password
- Login validation
- Stores login session using LocalStorage
- Redirects to Student Registration page after login

### 👨‍🎓 Student Registration Page
- Full Name
- Email
- Phone Number
- Gender selection
- Create Login ID
- Create Password
- College login verification required before access
- Form validation
- Responsive design

---

## 🔗 How It Works

1. College logs in using Login ID and Password.
2. Login status is stored in browser LocalStorage.
3. Student page checks whether college is logged in.
4. If not logged in → automatically redirects to College Login page.
5. If logged in → Student can register successfully.
